# WEB REST API

API Crud de Animais.

## Dependências

- npm
- node
- typescript
- lodash
- express

## Instalação

```bash
nvm use # definição da versão do node
yarn # ou npm install
```

## Execução

```bash
yarn start # ou npm start
```

## API Rotas

### GET

- /animals (query params: class, group & description)
- /animals/:id

### POST

- /animals

### PUT

- /animals/:id

### DELETE

- /animals/:id
